# Projektbeschreibung

`_radiant_Heliodata_SXR_0515` ist eine elektroakustische Forschungsstudie auf der Grundlage von Mess- und Ereignisdaten solarer Röntgenflares. Das Projekt behauptet nicht, einen wörtlichen „Klang der Sonne“ wiederzugeben. Stattdessen definiert es nachvollziehbare Beziehungen zwischen wissenschaftlichen Beobachtungen und hörbaren Parametern.

Untersucht wird, wie chronologisch geordnete Daten eine wahrnehmbare musikalische Morphologie bilden können, ohne auf eine konventionelle Melodie reduziert zu werden. Das Klangmaterial bleibt elementar: Sinusschwingungen, weißes Rauschen, Einzelimpulse, Hüllkurven, Filter und kurze Delay-Strukturen.

## Formmodell

Im Timeline-Modus wird das Beobachtungsfenster auf eine frei wählbare Aufführungsdauer komprimiert. Die Reihenfolge der Ereignisse bleibt erhalten. Wenn vollständige Zeitangaben vorliegen, bestimmen die Proportionen von Anstieg und Abfall die asymmetrische Klanghüllkurve.

Der maximale Röntgenfluss wird logarithmisch normalisiert, da die Klassen A, B, C, M und X aufeinanderfolgende Größenordnungen darstellen. Energiereichere Ereignisse erhöhen vor allem die spektrale und zeitliche Komplexität, nicht einfach die Lautstärke.

## Oberfläche

Die Max/MSP-Oberfläche ist als reduziertes Messinstrument gestaltet. Die monochrome Präsentationsansicht verbindet Transportsteuerung, Flussprofil, Ereignistabelle, Wellenform, Spektrum und Statusanzeige. Die Klangerzeugung bleibt im Patch sichtbar; die JavaScript-Engine übernimmt Datenimport, Prüfung, Normalisierung, reproduzierbare Zeitsteuerung und Display-Nachrichten.

## Umfang

Die erste Version ist bewusst kompakt. Später können dichtere GOES-Zeitreihen, CME-Beziehungen, Mehrkanal-Räumlichkeit und weitere visuelle Analysen ergänzt werden.
